YouTube Data API v3

필수
영상 검색과 메타데이터 조회에 사용됩니다. Google Cloud 프로젝트가 필요합니다.
Google Cloud Console에 접속합니다 → console.cloud.google.com
상단의 프로젝트 선택 → 새 프로젝트를 클릭하고, 프로젝트 이름을 입력한 뒤 만들기를 누릅니다.
좌측 메뉴에서 API 및 서비스 → 라이브러리로 이동합니다.
검색창에 YouTube Data API v3를 검색하고 클릭 → 사용 버튼을 누릅니다.
API 및 서비스 → 사용자 인증 정보로 이동합니다.
상단의 + 사용자 인증 정보 만들기 → API 키를 선택합니다.
생성된 API 키를 복사합니다. (선택: 키 제한에서 YouTube Data API v3만 허용하면 보안이 강화됩니다)

Gemini API (Google AI Studio)

필수 (LLM 3종 중 1개)
AI 대본 생성 + 영상 분석에 사용됩니다. Gemini를 선택하면 Google AI Studio 키도 겸용됩니다.
Google AI Studio에 접속합니다 → aistudio.google.com
Google 계정으로 로그인합니다.
좌측 메뉴 하단의 Get API key를 클릭합니다.
Create API key 버튼을 클릭합니다. 기존 Google Cloud 프로젝트를 선택하거나 새로 생성합니다.
생성된 API 키를 복사합니다.

Claude API (Anthropic)

필수 (LLM 3종 중 1개)
AI 대본 생성에 사용됩니다. 자연스러운 한국어에 강점이 있습니다.
Anthropic Console에 접속합니다 → console.anthropic.com
계정을 만들고 로그인합니다. (이메일 또는 Google 계정)
좌측 메뉴에서 API Keys를 클릭합니다.
Create Key 버튼을 클릭하고, 키 이름을 입력한 뒤 생성합니다.
생성된 API 키를 즉시 복사합니다. (이후 다시 볼 수 없습니다)
Settings → Billing에서 결제 수단을 등록하고 크레딧을 충전합니다. ($5~$10 권장)

💰 유료 전용. Sonnet: 입력 $3/100만 토큰, 출력 $15/100만 토큰. 대본 1개 생성 약 $0.01~0.03.

ChatGPT API (OpenAI)

필수 (LLM 3종 중 1개)
AI 대본 생성에 사용됩니다. ChatGPT를 선택하면 Whisper 음성 인식도 같은 키로 자동 사용됩니다.
OpenAI Platform에 접속합니다 → platform.openai.com
계정을 만들고 로그인합니다.
우측 상단 프로필 → API Keys (또는 좌측 메뉴 API Keys)를 클릭합니다.
+ Create new secret key를 클릭하고, 이름을 입력한 뒤 생성합니다.
생성된 키를 즉시 복사합니다. (이후 다시 볼 수 없습니다)
Settings → Billing에서 결제 수단을 등록하고 크레딧을 충전합니다. ($5~$10 권장)

Google Cloud Text-to-Speech

필수
AI 음성 합성에 사용됩니다. YouTube API 키와 같은 Google Cloud 프로젝트에서 발급받으면 편합니다.
Google Cloud Console에 접속합니다 → console.cloud.google.com
YouTube API를 만든 프로젝트를 선택합니다. (또는 새 프로젝트 생성)
API 및 서비스 → 라이브러리에서 Cloud Text-to-Speech API를 검색하고 사용을 누릅니다.
이미 YouTube용 API 키가 있다면 같은 키를 사용할 수 있습니다. 없다면 사용자 인증 정보 → API 키 만들기로 새로 발급합니다.
결제 메뉴에서 결제 계정을 연결합니다. (무료 할당량 사용을 위해서도 결제 계정이 필요합니다)
